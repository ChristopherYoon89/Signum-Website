Get Access to Our API

Our API provides structured access to our database of news articles, sources and categories collected from thousands of publishers. It is designed for developers and organizations that need programmatic access to news data with flexible filtering and ordering. All endpoints return normalized data that can be integrated into dashboards, research pipelines, monitoring tools or other applications.

As Signum News is still in its early stages, we currently provide API access via contact-based sales only. This approach allows us to scale our hardware infrastructure in a responsible manner. If you are interested in API access, please use the contact form and include a short description of your requirements.

For news articles, the API supports a wide range of filters to narrow results to specific use cases. You can query by source, category, tags or date ranges. Additional parameters allow filtering by algorithmic rating, user rating, click counts or source-level metrics such as average rating or article volume. Basic keyword matching on article titles is also available. Results can be ordered by publication date, algorithmic rating, engagement metrics, or source performance indicators, with reverse chronological order used by default.

The API also provides endpoints for exploring news sources. These allow filtering and ordering by source name, platform, user ratings or algorithm rating, which makes it easier to identify publishers that meet defined quality or platform criteria. This enables use cases such as building curated source lists, ranking publishers or analyzing coverage across platforms.

Category endpoints expose the available news taxonomy. Clients can retrieve categories by name and order them by identifier or alphabetically. This allows applications to dynamically discover available categories and build filtering interfaces without maintaining a separate taxonomy.
